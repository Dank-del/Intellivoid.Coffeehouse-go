# Intellivoid.Coffeehouse-go - Testing

### In order to run the tests, one needs to :
- make a `key.txt` in this directory and put their Coffeehouse API key there.
> Go to https://coffeehouse.intellivoid.net/dashboard and copy the access key.\
> That is your API Key for this test.
- Save the file, and run `go test` in this directory.

### Happy testing!
